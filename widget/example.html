Minutiae Visualization Widget

This page demonstrates the <mntviz-widget> Web Component. It can be embedded in any HTML page or Markdown-rendered document. Simply drop a fingerprint image and a .min file onto the widget below to visualize minutiae interactively.

Interactive Widget

Drag-and-drop an image file (.png, .jpg, .bmp) and one or more minutiae files (.min) onto the widget. You can also click to open a file browser. Multiple .min layers are supported with automatic color assignment.

After loading, use the mouse wheel to zoom and drag to pan. Hover over a minutia to see its coordinates, angle, and quality. Click on a minutia to see an expanded patch view. Use the toolbar buttons to add more files, reset the view, or clear everything.

Usage

Include the widget in any HTML page:

<script type="module" src="widget/mntviz-widget.js"></script>

<mntviz-widget style="width: 100%; height: 500px;"></mntviz-widget>

You can also configure the default minutiae color with the color attribute: 

<mntviz-widget color="#ff4444" style="width: 600px; height: 400px;"></mntviz-widget>

Programmatic API

const widget = document.querySelector('mntviz-widget');

// Load image
await widget.loadImage('path/to/image.png');

// Add minutiae from text
widget.addMinutiae(`
418 371 337 52
393 438 331 33
`, { name: 'probe', color: '#00ff00' });

// Clear
widget.clear();
